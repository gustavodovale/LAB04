# Laboratório Técnico 04: O Transformer Completo "From Scratch"

Este repositório contém a implementação da Etapa Final (Lab 04) do estudo sobre a arquitetura Transformer. O objetivo deste laboratório é atuar como uma "linha de montagem", integrando os módulos matemáticos construídos nos laboratórios anteriores (Scaled Dot-Product Attention, Causal Masking, Add & Norm e Feed-Forward Networks) em uma topologia coerente de **Encoder-Decoder**.

## Tecnologias e Bibliotecas Utilizadas
* **Linguagem:** Python 3.x
* **Biblioteca Matemática:** `numpy`
